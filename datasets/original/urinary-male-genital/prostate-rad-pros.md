# Prostate Cancers – Radical Prostatectomy Specimen

Datasets

Published Datasets

Urinary / Male Genital
>
Prostate Cancers – Radical Prostatectomy Specimen
Scope
The dataset has been developed for radical prostatectomy specimens for prostate carcinoma. Core biopsies and transurethral resection and enucleation specimens are dealt with in separate ICCR datasets. Rare urothelial carcinomas arising within the prostate are included in a separate ICCR dataset.

Publication History
3rd edition
– November 2024
Updated following the publication of the 2022 World Health Organization (WHO) Classification of Urinary and Male Genital Tumours, 5th edition.
This 3rd edition dataset was developed with the kind support of the International Society of Urological Pathology (ISUP).
SUMMARY OF CHANGES
• Updated to Union for International Cancer Control TNM8 staging, including errata published July 2024.
• Revised guide to harmonise with ICCR style for element titles and value list.
• Revised scope for clarity and to harmonise with ICCR style.
• Updated the ‘Clinical information’ element.
• ‘Specimen weight’ was changed to a non-core element.
• Renamed ‘Lymph nodes’ element ‘Lymph node dissection specimen(s)’ and updated.
• Updated the ‘Histological tumour grade’ element.
• Added a new element ‘Tumour growth patterns’ which incorporates the previous ‘Intraductal carcinoma of prostate’ element.
• Added a new element ‘Tumour quantification’ which incorporates the previous ‘Intraglandular extent’ element.
• Renamed ‘Urinary bladder neck invasion’ element ‘Microscopic urinary bladder neck invasion’.
• Updated the ‘Margin status’ element.
• ‘Lymphovascular invasion’ was changed to a core element.
• Updated the ‘Lymph node status’ element.
• Other minor updates to elements were also included.
• Revised all notes, including incorporation of the 2022 WHO Classification of Urinary and Male Genital Tumours, 5th edition.
