Eingeschlossen sind bösartige Erkrankungen am Rand der Schädelbasis. Auch neuroendokrine Neoplasien zählen dazu. Melanome, Lymphome, Sarkome, olfaktorische Neuroblastome und hämatolymphoide Tumoren sind nicht enthalten. Knochen- und Weichteiltumoren werden in separaten ICCR-Datensätzen behandelt. Halsdissektionen und Knotenexzisionen werden in einem separaten ICCR-Datensatz behandelt, und dieser Datensatz sollte gegebenenfalls zusammen verwendet werden. Dieser Datensatz ist für die Verwendung bei primären Krebsresektionen vorgesehen. Für Resektionen wiederkehrender Erkrankungen kann der Berichtsleitfaden pragmatisch verwendet werden, obwohl einige Datenelemente möglicherweise nicht anwendbar oder bewertbar sind. Updated following the publication of the 2024 World Health Organization (WHO) Classification of Head and Neck Tumours, 5th edition.
• Revised guide to harmonise with ICCR style for element titles and value list. • Revised scope for clarity and to harmonise with ICCR style. • Added a new element ‘Clinical information’, which incorporates the original element ‘Previous therapy’. • Updated the ‘Operative procedure’ element. • Updated the ‘Specimen(s) submitted’ element. • Updated the ‘Tumour site’ element. • Added a separate element ‘Tumour laterality’. • Removed the ‘Tumour focality’ element. • Added a new element ‘Block identification key’. • Updated the ‘Histological tumour type’ element. • Updated the ‘Histological tumour grade’ element. • Added a new element ‘Extent of invasion’, which incorporates the original element ‘Bone/cartilage invasion’. • Updated the ‘Margin status’ element. • Removed the ‘Coexistent pathology’ element. • Added a new element ‘Precursor lesion(s)’. • Updated the ‘Ancillary studies’ element. • Other minor updates to elements were also included. • Revised all notes, including incorporation of the most recent 2024 WHO Classification of Head and Neck Tumours, 5th edition.
